Arcade reconoSER - Piloto conectado a Google Sheets

Instrucciones:
1. Descomprime el ZIP completo.
2. Abre index.html con doble clic para probar localmente.
3. Para compartir por link y celular, sube todo el contenido a GitHub Pages.
4. Los registros se envían automáticamente a la hoja de Google configurada mediante Apps Script.
5. Si falla la conexión, también quedan guardados en el navegador y pueden exportarse desde el botón de registros.

Registros enviados:
- Ingreso autorizado
- Juego abierto
- Juego cerrado
- Juego recargado
- Juego completado
- Acceso a Plataforma reconoSER
